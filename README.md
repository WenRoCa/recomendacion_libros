# 📚 Sistema de Recomendación de Libros

Proyecto de Inteligencia Artificial que implementa un **sistema de recomendación basado en reglas de asociación** para una librería online. El sistema analiza patrones de compra y sugiere libros que suelen comprarse juntos.

---

## 🚀 Tecnologías

- Python 3.x  
- Pandas  
- mlxtend  

---

## 📊 Funcionalidades

- Análisis de compras: Descubre qué libros se compran juntos usando **Apriori**.  
- Reglas de asociación: Genera recomendaciones basadas en patrones de compra frecuentes.  
- Sistema de recomendación: Sugiere libros adicionales según el carrito actual del cliente.

---

## ▶️Ejecución

1. Clonar o descargar el repositorio:  
```bash
python recomendacion_libros.py

```
